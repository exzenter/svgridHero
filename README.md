An interactive canvas application that displays a grid of SVG symbols that rotate when you hover over them. Built with vanilla JavaScript and HTML5 Canvas.

## Features

- 40 different SVG symbols to choose from (plus, x, circles, arrows, stars, and more)
- Interactive rotation animations on hover
- Customizable grid size (rows and columns)
- Adjustable rotation angle and direction
- Multiple animation easing types (linear, ease-in-out, bounce, elastic, etc.)
- Customizable colors and stroke width
- Configurable animation speed
- Option to return symbols to original position when mouse leaves

## Usage

Just open `index.html` in your browser. Use the settings panel on the right to customize the grid, animations, and styling.

## Controls

- Adjust the grid size with the rows and columns sliders
- Change the symbol size percentage
- Set the rotation angle (0-180 degrees)
- Toggle animation on/off
- Adjust animation speed
- Choose from different easing functions
- Select different symbols from the dropdown
- Customize stroke color and width

The symbols will rotate based on which side of the symbol your mouse enters from, creating a dynamic interactive experience.
